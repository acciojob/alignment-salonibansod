Alignment

Create a webpage that describes the use of HTML tables. Follow the specific requirements provided below:

General Requirements

  1. Webpage Title: “HTML Assignment 3”
  2. Webpage Heading: Center-aligned “HTML Assignment #3”
  3. All tables should have the class “middle”

Table 1

Structure3 rows, 3 columns
ClassesMiddle row and rightmost column: “center” and “right”
IDsTable: “table-1”; Cells: “t11”, “t12”, “t13”

Table 2

Structure6 rows, 6 columns, 1st row in a single cell
IDsTable: “table-2”; First Cell: “t21”
AttributesFirst cell should have a colspan of 6

Table 3

StructureInclude an image; First and last rows: green and blue
IDsTable: “table-3”; Image Cell: “t-image”
ImageImage URL

Styling

  • Table Borders: 1px solid black; Spacing: 1px; Cell Padding: 1px
  • Background Colors: “green” and “blue” classes

Example

Here’s how your webpage should look:
